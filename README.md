# Cnode Lite
[cnodejs.org](https://cnodejs.org/api)社区的一个轻量型客户端，基于[vuejs](https://vuejs.org)，[vue-router](https://router.vuejs.org) and [vuex](https://vuex.vuejs.org)，使用[vue-cli](https://cli.vuejs.org)作为构建工具。

### 安装依赖项
```
npm install
```

### 运行开发环境
```
npm run serve
```

### 生产模式
```
npm run build
```

### 部署到Github Pages
```
npm run deploy
```
